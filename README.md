# Planetary Age Calculator

A Python program that calculates your age on different planets in the Solar System.  
By converting Earth years into the orbital periods of other planets, you can discover how old you’d be on Mars, Jupiter, Saturn, and beyond.

## Features
- Input your age in Earth years
- Get age equivalents on all planets in the Solar System
- Custom messages for Pluto and selected moons
